# Employee-Attrition-Predictor
An intelligent Machine Learning project designed to predict whether an employee is likely to leave a company based on factors such as education, city, experience, age, payment tier, and work history.
This project uses employee-related data to analyze patterns behind attrition and helps organizations improve employee retention strategies through predictive analytics.

📌 Project Overview
Employee attrition is one of the biggest challenges companies face. Losing experienced employees can lead to increased recruitment costs, productivity loss, and operational disruption.
This project aims to:


Analyze employee behavior and workplace patterns


Predict employee attrition using Machine Learning


Identify important factors influencing employee turnover


Support HR departments in making data-driven decisions



📊 Dataset Features
The dataset includes:


Education


Joining Year


City


Payment Tier


Age


Gender


Ever Benched


Experience in Current Domain


Leave Or Not (Target Variable)
